## Simple CRM Backend API

FastAPI və MySQL istifadə edərək hazırlanmış **sadə CRM backend API**.  
Bu layihə autentifikasiya, data ownership və əsas biznes məntiqini öyrənmək məqsədi daşıyır.

API **Swagger UI** üzərindən tam test edilə bilər.

---

## Məqsəd

Bu layihənin məqsədi:

- Backend əsaslarını öyrənmək
- SQLAlchemy ORM ilə database modelləşdirmə
- JWT autentifikasiya
- CRUD əməliyyatları
- İstifadəçi məlumatlarının ownership prinsipi ilə qorunması

---

## Texnologiyalar

- Python 3.11+
- FastAPI
- MySQL
- SQLAlchemy
- Pydantic
- JWT Authentication
- Docker (yalnız MySQL üçün)
